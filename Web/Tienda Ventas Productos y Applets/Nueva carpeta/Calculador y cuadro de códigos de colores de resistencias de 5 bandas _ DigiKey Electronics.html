Calculador de código de colores de resistencias de 5 bandas

Información sobre:

Esta herramienta se utiliza para decodificar información para las resistencias con conductores axiales en una banda de colores. Seleccione la cantidad de bandas y, luego, sus colores para determinar el valor y la tolerancia de las resistencias o vea todas las resistencias que Digi-Key tiene para ofrecerle.
