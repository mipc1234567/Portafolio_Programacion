CURSO DISEÑO DE PAGINAS WEB

INTRODUCCIÓN:

Esta página les permitirá aprender de una manera sencilla la construcción de páginas web para publicar por internet.

El curso está diseñado con todos los elementos necesarios como son explicaciones, videos, imágenes, audios, que permitían aclarar con facilidad lo necesario para llevar a cabo la construcción.

Tiene varios capítulos como son: botones, menús, pie de página, comentarios y codificación HTML respecto del tema que se esté tratando en este momento.

Se espera que al final del curso el estudiante haya aprendido a construir páginas web para publicar por internet.
