OBJETIVOS DEL OVI

Objetivo General:

Este OVI se construye para dar a conocer por medio de una página Web el curso de Diseño de sitios Web.

Objetivo Especificos 1:

Usar un programa para construir páginas web.

Objetivo Especificos 2:

Elaborar un borrador y un diseño Final de la página del curso.

Objetivo Especificos 3:

Elaborar, editar y redactar, textos y elementos finales de la información a publicar.

OBJETIVOS DEL CURSO

Objetivo General:

Este curso se realiza con la finalidad de aprender a diseñar de una manera rápida y eficaz un diseño de página web

Objetivos Especificos:

Estudiar las temáticas que brinda el curso de Diseño de sitios web para poder desarrollar la actividad.

Realizar investigaciones que correspondan al tema que estamos tratando.

Elaborar en html la página escogida por el grupo.
